1. ##### 加油站

   假设一条**环路**上有`n`个加油站，输入两个数组，`gas`数组表示每个加油站的汽油储量，而`cost`表示从当前加油站到下一个加油站的消耗容量，返回一个加油站编号作为起点，表示以这个加油站作为起点可以绕环路行驶一周

* 暴力法，首先计算出每个加油站到下一个加油站的汽油剩余量，并且将这个数组扩大一倍，然后遍历选取起点，判断是否合理

  ```js
  /**
   * @param {number[]} gas
   * @param {number[]} cost
   * @return {number}
   */
  var canCompleteCircuit = function (gas, cost) {
      /* 遍历数组，将每个加油站的汽油量减去开往下一站需要消耗的汽油得到新数组 */
      const rest = [];
      const len = gas.length;
      let sum = 0;/* 绕一圈的和 */
      for (let i = 0; i < len; i++) {
          rest[rest.length] = gas[i] - cost[i];
          sum += rest[rest.length - 1];
      }
      if (sum < 0) return -1;
      
      /* 将rest数组延长一倍，保证是一条环路 */
      rest.push(...rest);
  
      /* 在数组中选择起点 */
      outermost: for (let start = 0; start < len; start++) {
          /* 起点到下一站的剩余油量必须为正 */
          if (rest[start] < 0) {
              continue;
          }
          /* 开始计算 */
          let sum = rest[start];
          for (let i = start + 1; i < start + len; i++) {
              sum += rest[i];
              if (sum < 0) {
                  /* 不符合条件 */
                  continue outermost;
              }
          }
          return start;
      }
      return -1;
  };
  ```

* 暴力解法，就不通过加长一倍的方式来模拟一个圈了

  ```js
  /**
   * @param {number[]} gas
   * @param {number[]} cost
   * @return {number}
   */
  var canCompleteCircuit = function (gas, cost) {
      /* 暴力解法，使用while来模拟饶了一圈 */
      const len = gas.length;
      for (let i = 0; i < len; i++) {
          let rest = gas[i] - cost[i];
          let index = i + 1;
          /* 绕一圈 */
          while (rest > 0 && index % len !== i) {
              let cur = index % len;
              rest += (gas[cur] - cost[cur]);
              index++;
          }
          if (rest >= 0 && index % len === i) {
              return i;
          }
      }
      return -1;
  };
  ```

* 首先从第一个加油站开始计算绕一圈过程中的剩余汽油，得出最低点`min`以及最终剩余`sum`

  * 如果`sum<0`，说明不可能在不少油的情况下绕一圈
  * 如果`min>=0`, 说明从索引0开始即可顺利绕一圈
  * 否则`min`为负数，需要从后往前去填补`min`，一旦找到起点则返回

  ```js
  /**
   * @param {number[]} gas
   * @param {number[]} cost
   * @return {number}
   */
  var canCompleteCircuit = function (gas, cost) {
      /* 首先得到每经过一个加油站结余的汽油 */
      const len = gas.length;
      const rest = [];
      for (let i = 0; i < len; i++) {
          rest[rest.length] = gas[i] - cost[i];
      }
  
      /* 从0开始计算，得到整个数组到走完一圈的结余curSum */
      /* 以及记录过程中出现的最小的结余 */
      let sum = 0;
      let min = Infinity;
      for (let i = 0; i < len; i++) {
          sum += rest[i];
          min = Math.min(sum, min);
      }
  
      /* 如果sum小于0,说明不可能绕一周 */
      if (sum < 0) {
          return -1;
      }
      else if (min >= 0) {
          /* 如果期间没有出现负的结余 */
          return 0;
      }
  
      /* 从后往前找，找到能够填平min的索引 */
      for (let i = len - 1; i >= 0; i--) {
          min += rest[i];
          if (min >= 0) {
              return i;
          }
      }
  };
  ```

* 换个角度，首先判断整个一圈绕下来到底会不会少油，如果不会再找起点，从头开始找，计算剩油量，如果出现负数，说明前面的都不能作为起点了，再将下一个作为起点

  ```js
  /**
   * @param {number[]} gas
   * @param {number[]} cost
   * @return {number}
   */
  var canCompleteCircuit = function (gas, cost) {
      /* 首先得到经过每个加油站的剩余油量 */
      const len = gas.length;
      const rest = [];
      for (let i = 0; i < len; i++) {
          rest[rest.length] = gas[i] - cost[i];
      }
  
      /* 得到总剩油量 */
      let sum = rest.reduce((prev, cur) => prev + cur, 0);
      /* 如果小于0，说明永远无法绕一圈 */
      if (sum < 0) return -1;
  
      /* 开始找起点 */
      let curSum = 0;
      let startIndex = 0;
      for (let i = 0; i < len; i++) {
          curSum += rest[i];
          /* 如果遇到了负数，说明前面的都不能作为起点，从头来 */
          if (curSum < 0) {
              curSum = 0;
              startIndex = i + 1;
          }
      }
      return startIndex;
  };
  ```

2. ##### 分发糖果

   `n`个孩子站成一排，给定一个整数数组`ratings`表示每一个孩子的评分，现在按照要求给这些孩子分糖果

   - 至少每人一个
   - 相邻的两个孩子评分更高者得到更多的糖果

   返回需要准备的最少糖果数目

* 典型小气鬼，关键是要从左往右分，决定当前孩子分多少糖果的是前一个和后一个孩子的评分，如果左右孩子的评分比当前孩子的低，那就要保证当前孩子的糖果数比他们都多，如果某一边的评分大于等于当前孩子，那当前孩子就只需要给一个，这个过程中可能涉及递归，因为从左到右分的过程当中，如果右边孩子的分数低于左边孩子，那就需要先往右填，然后再回来

  ```js
  /**
   * @param {number[]} ratings
   * @return {number}
   */
  var candy = function (ratings) {
      /* 分糖果,传入当前编号 */
      function handleCandie(i) {
          /* 得到当前孩子的前一个评分和后一个评分 */
          let left = i - 1;
          let right = i + 1;
          let leftR = left >= 0 ? ratings[left] : Infinity;
          let rightR = right < len ? ratings[right] : Infinity;
          let cur = ratings[i];
  
          let rightIndex;/* 返回值，如果在过程中往右边填了，返回一直填到了哪个位置 */
  
          /* 根据前一个评分判断当前应该分多少糖 */
          let candiesFromLeft, candiesFromRight;
          if (leftR >= cur) {
              candiesFromLeft = 1;
          } else {
              candiesFromLeft = candies[left] + 1;
          }
          /* 根据后一个评分判断当前应该分多少糖 */
          if (rightR >= cur) {
              candiesFromRight = 1;
          } else {
              if (candies[right] === undefined) {
                  /* 如果右边还没分过 */
                  rightIndex = handleCandie(right);
              }
              candiesFromRight = candies[right] + 1;
          }
  
          /* 选择较大的数字分给当前的孩子 */
          candies[i] = Math.max(candiesFromLeft, candiesFromRight);
          return rightIndex || i;/* 返回 */
      }
  
      const len = ratings.length;
      /* 如果只有1个孩子 */
      if (len === 1) {
          return 1;
      }
      /* 记录糖果分配数组 */
      const candies = [];
      /* 遍历ratings，分配糖果 */
      for (let i = 0; i < len; i++) {
          i = handleCandie(i);
      }
      return candies.reduce((prev, cur) => prev + cur, 0);
  };
  ```

* 将结果分为四种可能性

  ```js
  /**
   * @param {number[]} ratings
   * @return {number}
   */
  var candy = function (ratings) {
      /* 分糖果,传入当前编号 */
      function handleCandie(i) {
          /* 得到当前孩子的前一个评分和后一个评分 */
          let left = i - 1;
          let right = i + 1;
          let leftR = left >= 0 ? ratings[left] : Infinity;
          let rightR = right < len ? ratings[right] : Infinity;
          let cur = ratings[i];
  
          let rightIndex;/* 返回值，如果在过程中往右边填了，返回一直填到了哪个位置 */
  
          /* 根据左右评分判断当前应该分多少糖 */
          if (cur <= leftR && cur <= rightR) {
              /* 如果当前评分在左右评分闭区间中 */
              candies[i] = 1;
          } else if (cur <= leftR && cur > rightR) {
              /* 如果当前评分比右边评分要高 */
              rightIndex = handleCandie(right);
              candies[i] = candies[right] + 1;
          } else if (cur > leftR && cur <= rightR) {
              /* 如果当前评分只比左边高 */
              candies[i] = candies[left] + 1;
          } else {
              /* 否则左右取较高者 */
              rightIndex = handleCandie(right);
              candies[i] = Math.max(candies[left], candies[right]) + 1;
          }
  
          /* 返回最终分到的最右索引 */
          return rightIndex || i;
      }
  
      const len = ratings.length;
      /* 如果只有1个孩子 */
      if (len === 1) {
          return 1;
      }
      /* 记录糖果分配数组 */
      const candies = [];
      /* 遍历ratings，分配糖果 */
      for (let i = 0; i < len; i++) {
          /* 并且更新索引，避免重复分配 */
          i = handleCandie(i);
      }
      return candies.reduce((prev, cur) => prev + cur, 0);
  };
  ```

* 分别从两边进行一遍遍历，不使用递归来同时保证左右的约束，省去了递归栈区占用的空间

  ```js
  /**
   * @param {number[]} ratings
   * @return {number}
   */
  var candy = function (ratings) {
      /* 前后两次遍历，先和左边孩子比较然后再与右边比较 */
      const len = ratings.length;
      /* 如果只有一个孩子 */
      if (len === 1) {
          return 1;
      }
      /* 数组记录分配的糖果,初始化第一个孩子的糖果为1 */
      const candies = new Array(len).fill(1);
  
      /* 从前往后确定 */
      for (let i = 1; i < len; i++) {
          let curR = ratings[i];/* 当前评分 */
          let prev = ratings[i - 1];
          /* 只有当前比前一个大的时候才需要递增 */
          if (curR > prev) {
              candies[i] = candies[i - 1] + 1;
          }
      }
  
      /* 从后往前加上右边孩子的约束 */
      for (let i = len - 2; i >= 0; i--) {
          let curR = ratings[i];
          let prev = ratings[i + 1];
          /* 如果当前评分比前一个要高 */
          if (curR > prev) {
              candies[i] = Math.max(candies[i], candies[i + 1] + 1);
          }
      }
  
      return candies.reduce((prev, cur) => prev + cur);
  };
  ```
